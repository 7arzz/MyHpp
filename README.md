# 💎 MyHPP - Professional Cost & Profit Manager

**MyHPP** is a premium, high-precision financial planning tool designed for small business owners and merchants. Calculate your **HPP (Harga Pokok Penjualan)**, determine ideal profit margins, and manage your product pricing with ease—all in a stunning, modern interface.

---

## 🌟 Key Features

- **📊 Precision HPP Calculation**: Automatically compute Cost of Goods Sold by factoring in raw materials, operational costs, and production quantities.
- **✨ Smart Pricing Recommendations**: Get instant selling price suggestions based on your target profit percentage.
- **📁 Product Management**: Add, edit, and organize your product catalog locally.
- **🔍 Quick Search**: Find your products instantly with the built-in search functionality.
- **🌙 Premium UI**: Experience a sleek, dark-themed interface with glassmorphism and smooth animations.
- **🔒 Data Privacy**: Your data stays on your device using Browser Local Storage—no backend or sign-up required.

## 📖 Cara Penggunaan (User Guide)

### 1. Menambah Produk Baru

Isi formulir "Tambah Produk Baru" dengan detail berikut:

- **Nama Produk**: Nama barang atau jasa Anda.
- **Biaya Bahan (Rp)**: Total biaya bahan baku yang digunakan.
- **Biaya Operasional (Rp)**: Biaya tambahan seperti kemasan, listrik, atau tenaga kerja.
- **Jumlah Produk (Pcs)**: Total unit yang dihasilkan dari biaya tersebut.
- **Target Profit (%)**: Persentase keuntungan yang Anda inginkan.

### 2. Melihat Daftar Produk

Setelah ditambah, produk akan muncul di daftar bawah. Anda akan melihat:

- **Total Biaya**: Gabungan biaya bahan dan operasional.
- **HPP / Pcs**: Modal dasar per unit produk.
- **Rekomendasi Harga Jual**: Harga jual ideal untuk mencapai target profit Anda.
- **Persentase Margin**: Margin nyata yang Anda peroleh.

### 3. Mengelola Data

- **Edit**: Klik ikon pensil ✏️ untuk memperbarui informasi produk.
- **Hapus**: Klik ikon tempat sampah 🗑️ untuk menghapus produk dari daftar.
- **Cari**: Gunakan kolom pencarian untuk menyaring produk berdasarkan nama.

## 🚀 Technical Setup (Developers)

If you want to run this project locally or contribute:

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository or download the source code.
2. Navigate to the project directory:
   ```bash
   cd MyHPP
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

### Build for Production

To create an optimized production build:

```bash
npm run build
```
